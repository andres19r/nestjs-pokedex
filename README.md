# Execute in dev mode

1. Clone the repository
2. Execute

```bash
yarn install
```

3. Must have the Nest CLI installed

```bash
npm i -g @nestjs/cli
```

4. Set up the database

```bash
docker compose up -d
```

## Used stack

- MongoDB
- NestJS
